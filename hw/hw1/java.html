Java

Java was developed by Sun Microsystems in 1982. This programming language is a class-based and object-oriented language that provides flexibility to developers. This flexibility allows programs written in this language to be able to run on virtually any machine. Other uses for this language include creating applets, web-based services, and basically any other type of application.
